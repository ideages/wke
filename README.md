# wke go binding

## build Environment

* OS: windows
* Go: latest i386 version go
* gcc: Mingw32-gcc

## Demo

* webcapture
```
go get github.com/xjdrew/wke/demo/webcapture && $GOPATH/bin/webcapture.exe
```

* interop between Go and JavaScript
```
go get github.com/xjdrew/wke/demo/interop && $GOPATH/bin/interop.exe
```
